Versionamento

O Git é um software que permite realizar versões de um projeto. O GitHub Desktop é um software que faz a ligação entre o Visual Studio Code, Git e GitHub.

O GitHub Desktop permite fazer a sincronia entre o repositório local e o repositório remoto


Commit

É um processo de confirmação de uma versão, ou seja, a cada alteração é possivel confirmar a alteração


Push

Este comando envia(sincroniza) as alterações realizadas no repositório local com o repositório remoto no GitHub
